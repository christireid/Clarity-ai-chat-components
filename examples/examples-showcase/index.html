Get Started in Minutes

Clone, install, and run any demo with simple commands

# Clone the repository
git clone https://github.com/your-org/clarity-chat.git
# Navigate to an example
cd clarity-chat/examples/analytics-console-demo
# Install dependencies
npm install
# Start the demo
npm run dev

API Keys Required

Each demo requires API keys from AI providers (OpenAI, Anthropic, Google). Copy .env.local.example to .env.local and add your keys.
