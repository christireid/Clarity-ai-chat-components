ðŸš€ Quick Start Guide

1. Clone the Repository

git clone https://github.com/your-org/clarity-chat.git
cd clarity-chat/examples

2. Navigate to an Example

cd model-comparison-demo
# Or: cd rag-workbench-demo
# Or: cd analytics-console-demo

3. Install Dependencies

npm install

4. Configure Environment

cp .env.local.example .env.local
# Edit .env.local and add your API keys

5. Start Development Server

npm run dev
# Or use PM2: pm2 start ecosystem.config.cjs

API Keys Required: Each demo requires API keys from AI providers. See each example's README for specific requirements.
