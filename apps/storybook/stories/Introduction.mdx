## ðŸ”§ Model Adapter System

Clarity Chat supports **model-agnostic** AI integration:

### Supported Providers
- **OpenAI** - GPT-4 Turbo, GPT-4, GPT-3.5 Turbo
- **Anthropic** - Claude 3 Opus, Sonnet, Haiku
- **Google AI** - Gemini Pro, Gemini Vision

### Features
- Unified streaming interface
- Cost estimation for all models
- Token usage tracking
- Easy provider switching (3 lines of code)

### Example
```typescript
import { openAIAdapter, ModelSelector } from '@clarity-chat/react'

// Use ModelSelector component
<ModelSelector
  models={allModels}
  value="gpt-4-turbo"

  showMetrics
/>
```
